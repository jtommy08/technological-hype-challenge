# 💻 Technological Hype App

## 📌 Description

Frontend application built with React, TypeScript, and Vite to display trending technology videos.

---

## ⚙️ Tech Stack

* React
* TypeScript
* Vite

---

## 📦 Installation

```bash
npm install
```

---

## ▶️ Running the Application

```bash
npm run dev
```

The app runs at:
http://localhost:5173

---

## 🔗 API Connection

The frontend fetches data from:
http://localhost:3000/api/videos

---

## 🧪 Notes

* Make sure the backend is running before starting the frontend
* Update API URL if backend port changes
