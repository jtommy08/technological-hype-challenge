# 🚀 Technological Hype Project

## 📌 Description

Fullstack application that displays trending technology videos.

* Backend built with NestJS
* Frontend built with React, TypeScript, and Vite

---

## ⚙️ Requirements

* Node.js
* npm

---

## 📦 Installation

Clone the repository:

```bash
git clone <your-repo-url>
cd technological-hype
```

---

## ▶️ Run Backend

```bash
cd api
npm install
npm run start:dev
```

The backend will run at:
http://localhost:3000

---

## 💻 Run Frontend

```bash
cd app
npm install
npm run dev
```

The frontend will run at:
http://localhost:5173

---

## 🔗 API Endpoint

```bash
GET http://localhost:3000/api/videos
```

---

## 🧪 Notes

* Start the backend before running the frontend
* Make sure ports 3000 and 5173 are available
